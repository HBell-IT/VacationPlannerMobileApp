# Vacation Planner Mobile Application

A native Android application designed to help users plan, organize, and manage vacation trips, excursions, and schedules.

---

##  Features

*  Create and manage vacation trips
*  Organize excursions and activities per trip
*  Edit and update trip details dynamically
*  Local data persistence using SQLite
*  Multi-screen Android UI navigation
*  Real-time updates to trip and excursion data
